# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-16

### Added
- Initial release of Arabic Lorem Ipsum Generator
- Core text generation engine with realistic Arabic grammar
- Multiple generation types: paragraphs, sentences, words, characters
- Theme-based vocabularies: general, technology, UI, design, business, mixed
- Output formats: plain text, HTML, JSON, Markdown
- Responsive web interface with RTL support
- Real-time text preview and statistics
- Copy to clipboard functionality
- Download in multiple formats
- REST API for programmatic access
- Preset management system
- Rate limiting for API
- Comprehensive documentation
- Unit tests with 70%+ coverage
- Deployment guides for multiple platforms

### Features
- Adjustable sentence length (min/max words)
- Custom keyword injection
- Punctuation control
- Word repetition prevention
- Option to start with classic Lorem Ipsum
- Real-time statistics display
- Mobile-responsive design
- Accessibility support

### Developer Features
- TypeScript for type safety
- React 18 with hooks
- Tailwind CSS for styling
- Vite for fast builds
- Express.js REST API
- Zod validation
- Jest testing framework
- ESLint code quality
